Model Preferences

The Model Preferences dialog box is used to adjust some global parameters and settings regarding the model.

  1. Model description - This field lets you add notes to your model file.
  2. Units - The consistent unit set to use. See the User's Guide, "Model database units" in "Mechanism Modeling".
  3. Modeling tolerance - The maximum allowed distance between the FE-node and its attached Triad. See the User's Guide, "Modeling tolerance" in "Mechanism Modeling".
  4. Gravitation - The gravity vector direction and magnitude. See the User's Guide, "Gravitation" in "Mechanism Modeling".
  5. Initial translational velocity - The initial translational velocity of all triads in the model. See the User's Guide, "Initial translational velocity" in "Mechanism Modeling".
  6. FE model repository - Switch between external and internal model repositories. You may also Change the external FE model repository. See the User's Guide, "Setting the FE model repository" in "Mechanism Elements".
  7. Overwrite existing reduced FE data - If enabled, a re-reduction of an FE part (due to changes in reduction options or changed FE model) will overwrite any existing reduction data for that part, instead of creating a new parallel sub-folder for the new reduction which is the default behavior.
  8. File for external function evaluation - If the toggle Use external function file is enabled, a file may be specified from which all External functions will get their value from, instead of expecting them from a calling program.

Changes to the Model description notes will be saved also when closing the Model Preferences dialog box, either by pressing the Close button or using the (×)-button in the window frame. Therefore, you may edit these notes at any time, also when you have results.

The File for external function evaluation is used only when the solver is run locally from the GUI itself. When the model is exported to a Digital Twin app for execution in the cloud, the Use external function file toggle is switched off.
