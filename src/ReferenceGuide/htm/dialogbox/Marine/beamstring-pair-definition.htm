Beamstring Pair Definition

The Beamstring Pair Definition tool is used to generate Free Joints between the Triads of two Beamstrings.

This tool is typically used when you have a pipe-in-pipe construction, and you would like the inner pipe to able to move inside the outer pipe. This tool will basically set up Free Joints between the overlapping Triads of the two pipes. The Free Joints will be assigned a Contact stiffness function that will limit the movement of the inner pipe with respect to the outer pipe.

The fields are as follows:

Name:Description:
First beamstringSpecifies the master beamstring, i.e., the master triads of the genereated Free joints are connected to this object. This can be a Subassembly of type Beamstring or Soil Pile.
Second beamstringSpecifies the slave beamstring, i.e., the slave triads of the genereated Free joints are connected to this object. This can only be a Subassembly of type Beamstring.
Contact stiffness functionSpecifies the contact stiffness function. This can be a function of type Stiffness-Translation or Force-Translation. It is possible to use an Adv. Spring Characteristics, Translation function as well.
Use radial springsSpecifies whether to use radial contact springs or not.

Please note that the two beamstrings must be overlapping, i.e., have the same number of triads, and the triads must all be in the same position. A tolerance value of 0.1 is used when checking if triads are co-located.
