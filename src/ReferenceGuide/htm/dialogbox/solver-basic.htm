Solver (Basic)

The Dynamics Solver dialog box (shown below) can be opened by choosing "Dynamics Solver (Basic Mode).." on the "Solve" menu. This is the basic version of the solver setup dialog box and contains the following settings:

  1. You can enable/disable a calculation of an initial static equilibrium configuration before starting the dynamics simulation.
  2. You can enable/disable a time history response analysis. If enabled, you can define the Time domain of the simulation through the Start time, Stop time and Time increment fields. If the Stop time toggle is disabled, the simulation will continue to run indefinitly or until it is stopped due to other reasons.
  3. You can enable/disable a quasistatic analysis. Either for the whole time domain, or up to the specified time.
  4. You can enable/disable natural frequency analyses and specify the number of eigenmodes to calculate in each analysis.
  5. You can enable/disable a frequency response analysis if the model contains loads defined as frequency domain input.
  6. Click the "Advanced" button to switch to the advanced version of the solver setup dialog box for more settings. The same button will take you back to the basic version if clicked again.
  7. Click the "Run!" button to start the dynamics solver.
