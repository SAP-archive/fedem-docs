Turbine Tower

The Tower Definition dialog box can be used to define a more complex tower represention, based on beam elements, in place of the default rigid representation that is generated when using the Turbine Definition dialog box.

  • D1 - Specifies the diameter of the tower at the top.
  • D2 - Specifies the diameter of the tower at the bottom.
  • T - Displays the tower height (not editable).
  • H1 - Specifies the height of the lower tower part of the tower.
  • H2 - Displays the height of the upper part of the tower (not editable).
  • Wall thickness - Specifies the thickness of the wall.
  • Material - Specifies the tower material.
  • Rho, E, v, G - Displays the properties of the chosen tower material.
  • N1 - Specifies the number of beam elements in the lower part.
  • N2 - Specifies the number of beam elements in the upper part.
