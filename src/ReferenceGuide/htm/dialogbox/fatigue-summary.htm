Fatigue Summary

The Fatigue Summary tool can be used to make an overview of fatigue for each event. You can do a fatigue summary analysis on any graph or set of curves in FEDEM. Just select a graph or set of curves, right click them, and choose "Fatigue Summary..". If you have multiple events in your model, then fatigue summary will be calucated for each event. You can adjust the probability values (Prob).
