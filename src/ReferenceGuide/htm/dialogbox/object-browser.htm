Object Browser

The Object Browser is used to provide detailed information about all objects in the model file. The Object Browser will display object details for the currently selected object in the model. You can also search for objects by typing in the search field and clicking the search button. The object browser will list all objects in the model if the search field is blank, or no object was selected when the object browser was activated. The Object Browser will search towards the user description field of all objects in the model.

Note: The ObjectBrowser does not display any object details for high-level and result objects.
