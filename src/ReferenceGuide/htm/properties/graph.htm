Graph

The Graph object is used to represent a set of Curves in a FEDEM model, to be displayed together in a graph view. You can create several Graph objects in a model, and an arbitrary number of Graphs can be displayed simultaneously in separate views in the workspace area.

The figure above illustrates the Graph properties, which consist of the following items:

Name:Description:
Title Specifies the title to be displayed above the graph.
X-axis label Specifies the label to be displayed on the abscissa axis.
Y-axis label Specifies the label to be displayed on the ordinate axis.
Font size: Specifies the font size to be used in the graph view for the Title, X-axis label and Y-axis label, respectively.
GridSpecifies the grid style. Possible choices are None, Coarse and Fine.
Legend Enables the display of the legends of each Curve in the Graph.
Autoscale Enables automatic scaling of the X- and Y-axes to accomodate the curves in the graph each time a new Curve is added to the Graph while its graph view is open, and also when opening the graph view associated with this graph.
Start time Specifies the beginning of the time interval to be used for all Curves plotting simulation results in the Graph.
Stop time Specifies the end of the time interval to be used for all Curves plotting simulation results in the Graph.
Use time interval Enables the use of the specified time interval [Start time,Stop time] for the Curves plotting simulation results. Use this option to speed up the process of loading data from disk if you have a very long time series, but only a small part of it contains interesting data.

See the section on "Graphs" in the "Postprocessing Results" chapter in the FEDEM User's Guide, for more information on creating and manipulating Graphs.
