Wind turbine controller

When the WTCtrl.dll plug-in is enabled in the Plug-Ins dialog box, the following set of User-defined function types are available in the Function property panel.

Low-pass filter

The figure above illustrates the properties of a Low-pass filter function, which consist of the following items:

Name:Description:
Argument xSelection of first function argument.
This should be the rotation velocity of the turbine generator.
Argument ySelection of second function argument.
This should be set to Time.
Corner frequency, f_c Specifies the corner frequency of the low-pass filter.

Torque controller

The figure above illustrates the properties of a Torque controller function, which consist of the following items:

Name:Description:
Argument xSelection of first function argument.
This should be the enforced rotation angle change on the wind turbine blades, i.e., a function of type Pitch controller.
Argument ySelection of second function argument.
This should be set to the filtered generator velocity, i.e., a function of type Low-pass filter taking the actual generator velocity as argument.
Argument zSelection of third function argument.
This should be set to Time.
Minimum pitch, region 3 Specifies the region 3 minimum pitch.
Rated speed Specifies the rated speed.
Rated power Specifies the rated power.
Transition speed region 1 and 1.5 Specifies the transition speed (region 1 and 1.5).
Transition speed region 1.5 and 2 Specifies the transition speed (region 1.5 and 2).
Torque constant region 2 Specifies the torque constant (region 2).
Rated slip percentage Specifies the rated slip percentage.
Max. torque, region 3 Specifies maximum torque (region 3).
Max. torque rate Specifies the maximum torque rate.

Pitch controller

The figure above illustrates the properties of a Pitch controller function, which consist of the following items:

Name:Description:
Argument xSelection of first function argument.
This should be the rotation angle of (one of) the wind turbine blades.
Argument ySelection of second function argument.
This should be set to the filtered generator velocity, i.e., a function of type Low-pass filter taking the actual generator velocity as argument.
Argument zSelection of third function argument.
This should be set to Time.
Reference speed Specifies the reference speed
Integral gain Specifies the integral gain.
Pitch for doubled power Specifies the pitch for doubled power.
Proportional gain Specifies the proportional gain.
Minimum pitch Specifies the minimum pitch setting.
Maximum pitch Specifies the maximum pitch setting.
Maximum pitch rate Specifies the minimum pitch rate.
