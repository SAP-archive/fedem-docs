Shaft

The Shaft object is a sub-assembly used to represent shafts in a wind turbine model.

The figure above illustrates the Shaft properties, which consist of the following items:

Name:Description:
Cross-sectionSpecifies the cross-section object.
MassDisplays the total mass of the shaft.
LengthDisplays the total length of the shaft.
Visualize 3DEnables or disables 3D visualization of the circular shaft.
Mass proportionalSpecifies mass proportional structural damping.
Stiffness proportionalSpecifies stiffness proportional structural damping.
Stiffness scaleSpecifies stiffness scale.
Mass scaleSpecifies mass scale.

The Shaft sub-assembly normally contains objects of type Beam and Triad in its model tree list.

Editing any of the Cross-section, Structural Damping and Scaling of Dynamic Properties fields, the values entered will apply to all the underlying Beam elements of the shaft, which can be verified by selecting any of the beam elements. But, if such a field is edited for one beam element such that the beams making up a shaft no longer have identical properties, the corresponding field in the Shaft property panel will be blank (no value displayed).

Note: The Visualize 3D toggle has a tri-state value. In addition to the usual ON and OFF states, it can also have a neither-ON-nor-OFF state, in which the corresponding ON/OFF settings on each underlying beam element are applied instead. This is usually the default setting for such tri-state toggles.
