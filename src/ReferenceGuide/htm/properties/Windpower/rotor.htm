Rotor

The Rotor object is a sub-assembly used to represent the rotor in a wind turbine model.

The figure above illustrates the Rotor properties, which consist of the following items:

Name:Description:
MassDisplays the total mass of the rotor.
Center of GravityDisplays the location of the rotor's center of gravity, in either Global coordinates or in Local coordinates with respect to the origin of the rotor assembly.

The Rotor sub-assembly contains a set of Blade assemblies in its model tree list. In addition, it normally contains a Hub Part, a set of Pitch Revolute Joints (one for each blade), and one Rigid Joint defining the connection to the Main Shaft of the driveline. Finally, there is a set of Triads (one for each joint).
