Turbine

The Turbine object is a sub-assembly used to represent a wind turbine model in FEDEM.

The Turbine sub-assembly with all its sub-components are created from the Turbine Definition dialog. In its model tree list, it typically contains a Rotor, a Nacelle and a Tower, which again contain further sub-components themselves. In addition, it contains some Beam cross sections and Materials used by the Shafts, the Yaw Revolute Joint that connects the Nacelle and Tower, and a Free Joint connecting the tower to the ground. The latter is represented by a single Triad that may be reattached to a substructure, if the turbine model is to be placed on a flexible (or floating) foundation.

Please see the chapter on "Wind Power Modeling" in the FEDEM User’s Guide, for more information on wind power modeling and simulation in FEDEM.

Name:Description:
MassDisplays the total mass of the turbine.
Center of GravityDisplays the location of the turbine's center of gravity, in either Global coordinates or in Local coordinates with respect to the origin of the turbine assembly.
PositionSpecifies the position of the the turbine assembly.
OrientationSpecifies the orientation of the turbine assembly.
Wind reference pointSpecifies a Triad and a vertical offset that together define the reference point (origin) for the wind field.
Advanced topology settingsSpecifies some objects whose locations are required in the calculation of the aerodynamic forces on the turbine.

Normally, there is no need to change the Wind reference point and Advanced topology settings. They are set automatically when generating a turbine model from the Turbine Definition dialog. However, sometimes the settings can become lost when modifying the components of the turbine. If you for example were to delete the tower of the turbine, you would see that the Reference triad and the Yaw point triad will be unassigned. If you later add a tower manually, you would have to reset these fields to proper values.
