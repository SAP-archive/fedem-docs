Reference Plane

The Reference Plane object is used as a representation of the ground in a FEDEM model, and also as a pure visual reference. It is visualized as a shaded square in the center of the Modeler window when you start a new modelling session.

The figure above illustrates the Reference Plane properties, which consist of the following fields:

Name:Description:
Position Specifies the position of the reference plane.
Orientation Specifies the orientation of the reference plane.
Height Specifies the height of the reference plane.
Width Specifies the width of the reference plane.

The Position and Orientation fields have similar interpretation as as in the Origin tab of the Triad property panel.
