FrictionSpecifies the friction in the Free joint. Joint DOFSpecifies which of the six joint DOFs that shall receive the friction force or moment. The effective normal load in the friction is computed from the two joint DOFs that are orthogonal to the selected DOF. See the FEDEM Theory Guide, Section 6.5 "Joint Friction". Set All FreeEnables you to set the Constraint type of all joint DOFs to Free. Set All FixedEnables you to set the Constraint type of all joint DOFs to Fixed.
