Beam Cross Section

The Beam cross section object is used to describe the cross section properties of a Beam element.

The two figures above illustrate the Structural properties for the cross section types Pipe and Generic, respectively, and consist of the following items:

Name:Description:
Cross section type Specifies the cross section type. We currently support “Pipe” and “Generic”.
MaterialSpecifies the Material property object that this cross section is to be associated with.
Definition (Pipe) Depending on the cross section type, you will have various fields for defining the cross section geometry. For Pipe sections, you specify the outer (Do) and inner (Di) diameters.
Definition (Generic) For Generic cross sections, you specify axial stiffness (EA), bending stiffnesses (EIyy and EIzz), torsional stiffness (GIt), mass per unit length (ρA), and torsional moment of inertia (ρIp). Note that these property parameters also include the neccessary material information. Thus, no Material object is needed for Generic cross sections.
Shear reduction factors These values (ky and kz) specify the ratio between the effective transverse shear area and the actual cross section area, in the local y- and z-axis directions, respectively. Refer to text books on structural mechanics on how these parameters depend on different cross section shapes. If these values are set to zero, the transverse shear deformation is ignored in the FE model (infinite shear stiffness).
Shear stiffness For Generic cross section, you specify the shear stiffness (GAs,y and GAs,z) directly, instead of the shear reduction factors. Zero (or negative) values in these fields will be interpreted as infinite shear stiffness (no shear deformation).
Shear center offset These fields (sy and sz) allow you to specify the local position of the shear centre of the cross section w.r.t. its elastic centre (or the neutral axis). The shear centre defines the point through which a transverse load must be act in order to avoid torsional deformation in the beam element. For symmetric cross sections the shear centre conicides with the elastic centre and the values 0.0 are appropriate.
Dependent properties These properties (cross section area, A, and moments of inertias, Iyy, Izz and Ip) are updated automatically based on the specified geometry parameters. However, if the Break dependency toggle is enabled any of the dependent properties may be overridden manually.
