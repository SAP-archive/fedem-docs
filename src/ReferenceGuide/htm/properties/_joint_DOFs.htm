The available options here depend on the selected Constraint Type, which can be as follows:
  • Free - This DOF is free to move and no constraint is applied. This is the default setting.
  • Fixed - This DOF is fixed and can not be moved at all. It is eliminated from the system of equations (condensed out).
  • Prescribed - This DOF can be assigned a time-dependent prescribed motion, and is thus condensed out from the system of equations.
  • Spring-Damper - This DOF is free to move, but a spring and/or damper may be applied to constrain the motion.

The figure above illustrates the Free joint DOF properties, and consist of the following items:

Name:Description:
Additional BCWhen this toggle is enabled, the movement in this joint DOF is fixed during the initial equilibrium analysis and optionally also in the eigenmode analysis. (See the Eigenmode tab, and the FEDEM Theory Guide, Section 7.8 "Quasistatic equilibrium".)
Load magnitudeYou can apply a Load in this joint DOF. This will be a torque or a force depending on whether this DOF is a rotation or translation.
Initial velocityYou may specify a velocity that should be applied as initial condition in this joint DOF, i.e., the actual velocity at the start time of dynamics simulation.
Length/Angle in modelThis field shows the initial value of this joint DOF as modeled. For rotational DOFs the value can be edited to set a different initial rotation state. The 3D modeller view will then update instantly, showing the new orientation of the joint symbol.
Apply in frequency domainIf a Function is specified in the Load magnitude field, this toggle will enable the load to be applied in the frequency domain instead of the time domain, if the Frequency response analysis toggle is enabled in the Solver setup dialog.

The Fixed joint DOF properties are the same as for Free DOFs, except that the Additional BC toggle here is always inactive, and the Load magnitude and Initial velocity fields, as well as the Apply in frequency domain toggle, all are absent, as illustrated in the figure above.

The figure above illustrates the Prescribed joint DOF properties, and consist of the following items:

Name:Description:
Prescribed quantityYou may choose whether you want to prescribe the Deflection, the Velocity or the Acceleration for the joint DOF.
Initial velocityYou may specify a velocity that should be applied as initial condition in this joint DOF, i.e., the actual velocity at the start time of dynamics simulation.
Length/Angle in modelThis field shows the initial value of this joint DOF as modeled, and has the same interpretation as when the joint DOF is Free (see above).
Constant length/angle, Constant displacement/rotation These buttons and fields work together to allow you to set a constant prescribed value of the joint DOF during the dynamics simulation. You can choose to enter the constant length/angle either as an absolute value, or relative to the Length/Angle in model, if the Prescribed quantity is set to Deflection. For prescribed Velocity or Acceleration, you enter the constant velocity/acceleration that should be applied in the first of these two fields (the other field is inactive).
Length/Angle changeYou can prescribe the evolution of the motion during the simulation through a function. The function controls the change in the DOF valie relative to the Constant length/angle, or the change of the velocity/acceleration relative to the Constant velocity/acceleration, in case Velocity or Acceleration is prescribed.
Apply in frequency domainIf a Function is specified in the Length/Angle change field, this toggle will enable the prescribed motion to be applied in the frequency domain instead of the time domain, if the Frequency response analysis toggle is enabled in the Solver setup dialog.

Note that if you prescribe a length/angle that differs from the Length/Angle in model, this difference will be accounted for in the very first iteration of the dynamics simulation and thus lead to a dynamic shock effect. However, when the initial Static equilibrium analysis is enabled, the force due to this difference is taken as a pure static load and the transient shock should be avoided.

The figure above illustrates the Spring-Damper joint DOF properties, and consist of the following items:

Name:Description:
Additional BCWhen this toggle is enabled, the movement in this joint DOF is fixed during the initial equilibrium analysis and optionally also in the eigenmode analysis. It has the same interpretation as when the joint DOF is "Free" (see above).
Load magnitudeYou can apply a Load in the joint DOF, similarly as when the joint DOF is Free (see above).
Initial velocityYou may specify a velocity that should be applied as initial condition in this joint DOF, similarly as when the joint DOF is Free (see above).
Length/Angle in modelThis field shows the initial value of this joint DOF as modeled, and has the same interpretation as when the joint DOF is Free (see above).
Constant stress free length/angle, Constant deflection These buttons and fields work together allowing you to introduce prestress in the spring, by setting a constant stress free length/angle different from the Length/Angle in model. You can chose to enter this property either as an absolute value, using the Constant stress free length/angle option, or relative to the Length/Angle in model by selecting the Constant deflection option.
Stress free length/angle change You can select a Function to change the stress free length/angle of the spring during the simulation. The value of the function will be used as an addition to the constant stress free length/angle defined above.
StiffnessIn this field either a constant spring stiffness can be entered, or you can select a defined spring characteristic from the pull-down list.
Spring properties; ScaleThe spring force or torque can be scaled by a function. This can for instance be used to switch the spring on and off during the simulation. When no function is selected the scale is set to 1.0.
Damping coefficientIn this field either a constant damping coefficient can be entered, or you can select a defined damper characteristic from the pull-down list.
Damper properties; ScaleThe damper force or torque can be scaled by a function. This can for instance be used to switch the damper on and off during the simulation. When no function is selected the scale is set to 1.0.
Use deformational velocity This option is available if the damper is acting together with a spring with a forced change in its stress free length. The option enables the usage of the deformational velocity of the connected spring, when evaluating the damper. The deformational velocity is the spring velocity without the velocity component coming from a forced change in the stress free length.

Note that if prestress is introduced in a joint DOF spring, it will be accounted for in the very first iteration of the dynamics simulation and may thus lead to a dynamic shock effect. However, when the initial Static equilibrium analysis is enabled, the prestress force is taken as a pure static load and the transient shock should be avoided.
