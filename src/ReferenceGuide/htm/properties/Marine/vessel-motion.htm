Vessel Motion

The Vessel Motion object is used to represent the motion of a rigid floating marine vessel through a Response Amplitude Operator (RAO). An RAO expresses the motion of a point on the vessel as a function of the wave height and frequency at the same reference point.

The figure above illustrates the Vessel Motion properties, which consists of the following items:

Name:Description:
RAO fileSpecifies a file containing the RAO table(s) describing the vessel motion as functions of wave elevation and frequency.
Wave functionSpecifies the Wave Function defining the wave height, angular frequency and phase delay, used by the six motion Functions associated with the vessel motion object. Normally, this should be the same function as specified as the Wave function in the Sea Environment dialog, but is not necessarily so.
Wave directionSpecifies the angle (in degrees) between the local X-axis of the Vessel Triad and the incomming wave direction. A zero wave direction means that the waves propagate in the direction of the negative local X-axis of the Vessel Triad and the value 90.0 means that the waves propagate in the direction of the negative Y-axis. A separate RAO table is associated with each of the direction angles available in the specified RAO file.
Motion scaleSpecifies a Function to be used as a scaling on the six motion functions associated with the vessel motion. This can be used to ramp up the RAO motions in a smooth manner, to avoid undesired startup trascients. When a function is specified here, six new motion functions are generated, where each one is defined as the product of the original motion function and the scaling function.

Please see the chapter on "Marine Modeling" in the FEDEM User’s Guide, for more information on the RAO file format and how the motion functions associated with the vessel motion object are defined.
