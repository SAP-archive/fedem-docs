ConstraintThis column shows the constraint type assigned to each joint DOF. LoadThis column shows the applied load in each joint DOF. Model lengthThis column shows the modeled length/angle in each joint DOF. Init disp.This column shows the initial deflection in each joint DOF. Length changeThis column shows the function, if any, that controls the change of length/angle of each joint DOF. If the Constraint type is set to Prescribed, this change is enforced directly on the joint DOF. However, if the Constraint type is set to Spring-Damper, this function controls the Stress free length/angle change of the associated joint DOF spring. Init vel.This column shows the initial velocity applied to each joint DOF. SpringThis column shows the spring characteristic assigned to each joint DOF; either a numerical value defining a constant spring stiffness, or a description of the spring characteristics used as a nonlinear stiffness- or force-deflection relationship. Spr. scaleThis column shows the function that is used to scale the force developed in the spring of each joint DOF. Empty if no scaling. DamperThis column shows the damper characteristic assigned to each joint DOF; either a numerical value defining a constant damper coefficient, or a description of a function used as a nonlinear coefficient- or force-velocity relationship. Dmp. scaleThis column shows the function that is used to scale the force developed in the damper of each joint DOF. Empty if no scaling.
