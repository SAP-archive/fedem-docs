Curve

The Curve object is used to represent a set of data points which can be plotted as a curve in a two-dimensional graph view. The data points can be simulation results, calculated from explicit functions, or read from a specified file. The most commun use of Curve objects is to plot simulation results against time as the simulation evolves, or to plot them after the simulation has finished in a post-processing session.

See the chapter on "Postprocessing Results" chapter in the FEDEM User's Guide, for more information on creating and manipulating Curves.

Data tab

The figures above illustrate the Data properties for the Curve, which consist of the following items:

Name:Description:
Legend Specifies the legend to be used for the curve in the graph view.
Auto legend Enables automatic generation of the legend text based on the curve definition.
Source Chooses the type of data source for the curve.
X Axis Defines the x-axis data source for a From RDB curve.
Y Axis Defines the y-axis data source for a From RDB curve.
Export curve automatically Enables automatic export of this curve when the simulation is finished.
Function of A,B,C Specifies a mathematical expression with up to 10 variables (named A,B,...,J) for a Combined curve. Each variable represents another Curve. The expression obeys the same syntax rules as for Math expression Functions.
A: / B:Specify the Curves to be represented by each variable in a Combined curve. The number of such fields is automatically set depending on the number of variables in the mathematical expression.
FileDisplays the data file name when the curve is plotting data From file.
Browse... Selects the data file to use for a From file curve.
ReloadReloads the curve data From file. Use this button if you change the content of the file while the graph view plotting the file is open.
Column/channel Displays the name of the selected named channel (or column) if the chosen file contains multiple channels.
Select... Specifies which channel or column to read data from, if the chosen file contains multiple channels/columns.
Start x / Stop x Specifies the end points of the argument domain, if the curve is plotting an Internal function.
IncrementSpecifies a constant sampling rate for the Internal function to be plotted.
Auto IncrementEnables automatic detection of the curve point resolution, depending on the type of the Internal function to be plotted.
Function Specifies the Internal function to be plotted.

Appearance tab

The figure above illustrates the Appearance properties of the Curve, which consist of the following items:

Name:Description:
Curve type Specifies the visualization style for the curve (Lines, Dots, or Invisible).
Curve thickness Specifies the thickness level of the curve (0 to 4).
Curve color Specifies the color of the curve from a set of pre-defined colors.
Symbol type Specifies the symbol to be displayed at each data point in the curve.
Symbol size Specifies the size of the symbol.

Curve Statistics tab

The figure above illustrates the Curve Statistics properties of the Curve, which consist of the following items:

Name:Description:
Calculate Click this button to recalculate the statistics of the curve data.
RMSDisplays the root mean square.
MeanDisplays the mean value.
Std. dev.Displays the standard deviation.
IntegralDisplays the integrated value (i.e., the area under the curve).
MaxDisplays the maximum value.
Mindisplays the minimum value.
Use scaled/shifted Enables the use of the Scale and Shift values when calculating the curve statistics.
X Axis Domain Specifies the start and stop points of the X-axis domain to be used in the calculations. The Entire button enables the use of all data points of the curve.

Scale and Shift tab

The figure above illustrates the Scale and Shift properties of the Curve, which consist of the following items for each of the two axes:

Name:Description:
Scale Specifies a scaling factor for the curve data values.
Shift values to zero Enables a constant shift of the data values such that the first curve point has a zero value.
Additional shift Specifies a constant shift to be applied in addition to the zero-out operation, if enabled.

Fourier Analysis and Differentiation tab

The figure above illustrates the Fourier Analysis and Differentiation properties of the Curve, which consist of the following items:

Name:Description:
On/Off Enables the replacement of the plotted curve by its discrete Fourier transform. This option is avalable only if the curve uses Physical time on its X axis.
Time domain Specifies the time domain to be used in the Fourier transformation.
No 0 Hz component Enables cancellation of the zero Hz component in the Fourier transform.
Use sample rate Specifies the sampling rate to be used in the Fourier transformation, in case the curve data points have a non-uniform distribution in time.
Differentiate Enables the replacement of the plotted curve by its derivative.
Integrate Enables the replacement of the plotted curve by its integral.

Rainflow and Fatigue tab

The figure above illustrates the Rainflow and Fatigue properties of the Curve, which consist of the following items:

Name:Description:
Standard Specifies the fatigue standard to be used for the calculations.
S-N curve Specifies the S-N curve from the specified standard.
Stress range threshold Specifies the smallest stress range magnitude that will be accounted for in the stress cycle counting (rainflow analysis).
Damage Displays the calculated damage value.
Life Displays the calculated life value.
Life unit Specifies the units for the displayes life value (Repeats, Hours or Days).
Time Interval Specifies the time domain to be used in the Fatigue calculation.
Show rainflowEnables the replacement of the plotted curve by the results of the rainflow analysis, showing the magnitudes of the stress ranges that are the basis of the subsequent damage calculation.
