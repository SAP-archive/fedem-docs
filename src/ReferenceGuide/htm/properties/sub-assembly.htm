Subassembly

The Subassembly object is used to group a complex FEDEM model into sub-models in a hiearchical manner.

A Subassembly may contain sets of any other structural objects (Beams, Parts, Triads, etc.), including other Subassemblies, in its model tree list.

The figure above illustrates the Subassembly properties, which consist of the following items:

Name:Description:
Subassembly model fileSpecifies a model file for the Subassembly. A separate model file may be assigned to the Subassembly by using the Browse button.
MassDisplays the total mass of the Subassembly.
Center of GravityDisplays the location of the Subassembly's center of gravity, in either Global coordinates or in Local coordinates with respect to the origin of the Subassembly.
PositionSpecifies the position of the Subassembly.
OrientationSpecifies the orientation of the Subassembly.

If a separate model file is assigned to a Subassembly, all sub-components of that assembly are stored in the specified file when the model is saved. It is then possible to reuse this Subassembly in another model by importing it into an existing model. If the Subassembly model file field is left empty, the elements of the Subassembly are stored in the main model file together with the rest of the model.
