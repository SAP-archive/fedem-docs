Animation

The Animation object is used visualize kinematic motions and structural results of the model. This includes Part motion and deformation, and various scalar result quantities visualized as color contour plots. You can create several Animation objects in a model with different options for each. The created Animations can then be loaded into the Modeler window one at a time.

The figures above illustrate the Animation properties, which consist of the following items:

Name:Description:
TypeYou can switch between three different animation types:
- Time History: Visualizes time history results. This can be the rigid body component of Part motion calculated in the dynamics solver, or recovered stress, strain and the deformational part of the Part motion.
- Eigen Mode: Visualizes the shape of a system eigenmode at a specific point in time. It can also be used to visualize the component mode shape of an FE Part, or the free-free mode shape of a reduced FE Part.
- Time Summary: Visualizes accumulated structural results within a specified time interval. These types of results are produced by running the Strain Coat Recovery Summary.
Load face contoursIf this toggle is enabled, the results specified in the Contours tab are loaded and displayed as color contours on the element faces of the mechanism assembly.
Load line contoursThis toggle controls whether or not the results specified in the Contours tab are displayed as color contours on the FE-mesh lines.
Load deformationsIf this toggle is enabled, Part deformations are loaded and displayed, if such results are present. If this toggle is disabled, or stress recovery has not been performed, only the rigid body motion of each Part is displayed.
Load AnimationClick this button to actually load the animation using the current property settings.
Time WindowYou can choose to display the complete duration of the simulation or a specific time interval. If you select Time Interval, specify the values for the interval's Start and Stop time.
Animation FramesThese options are useful if you specified different time steps for the dynamics simulation and the stress recovery calculation. Enabling For every time step loads deformation results and color contour values for all the time steps for which such results have been calculated within the time window specified. For all the other time steps only the rigid body Part motion will be displayed. If you enable Only for requested results, only those time steps having deformation/contour results will be loaded.

Name:Description:
Contour valueThese options are used to select the results that will be displayed as color contours.
- Result class: Selects the FE entities for which you want to show color contours. You can select either Node, Element (one single result for each element), or Element node (one result value for each node within an element).
- Result: Specifies which type of result to display in the animation. The options available here depend on the Result class setting, and on the results currently available in the results database.
- Operation: Specifies the scalar value to extract from the selected result, if that is a non-scalar quantity. The operations available depend on both the Result class and Result settings.
Result SetThis setting is used to distinguish between multiple results for the specified FE entity, including results from different layers within an element or multiple deformation results on a node.
- By operation: Consolidates the result set of the same type using the selected operation (Average, Maximum, Minimum, etc.).
- By name: Loads and displays only the result sets with the specified name. Elements of an FE Part without that result set will be rendered gray, whereas complete Parts without that result set will remain unchanged.
AveragingThese options control how a continuous contour plot is obtained when the underlying results are discontinuous across the finite elements. They are available only when the selected Result class is Element node.
- Operation: Selects the averaging operation to use (Average, Max, Min, etc.), or select None if averaging is not wanted.
- on: Averaging on Node calculates a single value for each node in the FE model, based on the selected Operation and the values specified in the Contour Value settings. Averaging on Element calculates a single value for each element in the same way.
- Max shell average angle: Specifies an angle above which the averaging between shells stops. The angle is used as a tolerance when comparing the globalized coordinate systems of the elements in question.
- Average across element type: Toggles averaging of contour values across element type borders.
Multiple Face ResultsSpecifies the averaging behavior across elements that interface on a surface (such as stacking of elements with the same size and shape but different thicknesses). You can either specify an averaging operation, or select a named element group containing the result to be shown.
Export animation automatically to VTFEnables export of the animation to a VTF-file after a batch stress recovery, if FEDEM is run with the command-line option -exportAnimation.

Name:Description:
Mode SelectionThese options are available for Eigen Mode animations:
- Select between System modes, Component modes of part and Free-free modes of reduced part.
- Time: For System modes, this pull-down list shows the times at which an Eigenmode solution is specified in the Eigenmode tab of the Dynamics Solver Setup dialog.
- Part: For Component modes of part or Free-free modes of reduced part, this pull-down lists all FE Parts in the model. It also have an entry (All parts) to enable simultaneous mode shape animations for all FE Parts.
- Mode: Shows either the modes specified in the Eigenmode tab, the component mode numbers in the Reduction Options tab of the Part property, or the free-free mode numbers for the reduced part, depending on the selected mode type in the first pull-down list.
- Scale: Specifies a scaling factor to exaggerate the shapes during the animation.
Frame GenerationThese options set the number of animation frames and the duration of the animation:
- Frames per cycle: Entering a higher value increases the continuity of the animated motion.
- Length: Specify either a Time, No. of cycles, or Until % damped to limit the duration of the animation. These fields are available only for System modes. For the other two options the duration of the animation will always be equal one full cycle.

See the section on "Animation" in the "Postprocessing Results" chapter in the FEDEM User's Guide, for more information on manipulating Animations.
