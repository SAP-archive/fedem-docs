FrictionSpecifies the friction in the Ball joint. Joint DOFSpecifies which of the three joint DOFs that shall receive the friction moment. The effective normal load in the friction is then computed from the other two joint DOFs that are orthogonal to the selected DOF. See the FEDEM Theory Guide, Section 6.5 "Joint Friction". Set All FreeEnables you to set the Constraint type of all joint DOFs to Free. Set All FixedEnables you to set the Constraint type of all joint DOFs to Fixed.
