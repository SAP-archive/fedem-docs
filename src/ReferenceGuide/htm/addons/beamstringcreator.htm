Beam String Creator

This tool is used to create beam strings as straight lines or in the shape of a catenary-curve. The catenary shape can include the effects of buoyancy, and tension caused by the shape can be included in the generated beam string.

Please note that you must define cross sections in FEDEM before using this tool.

A beam string contains one or more segments, with various dimensions and level of discretization/number of beam elements.
Each segment corresponds to one row in the applications table.
To add a segment you must select a cross section from the drop-down menu in the bottom row, specify the total length for your segment, and the length of the beams the segment should be made of.

Straight beam string

The default mode for the Beam String Creator creates a straight beam string starting from the position specified in the End 1 input box, shooting straight out in the direction specified by the vector in the Direction input box.

Catenary Curve

When the "Catenary Curve" check-box is checked, the Beam String Creator will attempt to generate the beam string in a catenary shape, that is in the shape of a hanging chain.
The Direction input box will now be gray, and a second end-point must instead be specified in the End 2 input box. The beam string will "hang" between these two points as long as the distance between them is shorter than the cumulative length of the beam strings segments.
If the application succeeds in finding a catenary shape, a plot of the beam string will be shown in the window below the segment-table.

Buoyancy

If the "Buoyancy" check-box is checked, buoyancy effects will be included in the catenary calculation.
The buoyancy calculation is based on the buoyancy diameter Db for each segments cross section, and a water density of 1025kg/m^3.

Tension

When laid out in a catenary shape, the beam string will by default have no tension in this shape. The separate beams are simply rigidly connected to each other.
With "Tension" toggled on, joints will be placed between each pair of beams, and rotational springs with tension corresponding to the angle between consecutive beams will be added. Calculated tension is based on the E-modulus for each cross sections material, and the second moment of area for the cross section. Cross sections are assumed to be circular.
Note that the tension effect is not included in the calculation of the catenary shape, and will thus only be a reasonable approximation for beam strings with low stiffness. This means that the beam string is not in equilibrium in its initial state.
