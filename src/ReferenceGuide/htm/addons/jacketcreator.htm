Jacket Creator

The Jacket Creator is a demo-application used to illustrate how a relatively complex beam structure can be scripted using the FEDEM COM-interface.

How to use it

  1. Define the four corner points for both the bottom and top level of the jacket.
    You can do this by either entering coordinates for each point, or by setting the "Bottom width" and "Top width" parameters to create square planes.
  2. Enter the cross section properties(outer and inner diameter) for the legs, x-braces and horizontal braces.
  3. Set the desired "Level heights". Each height corresponds to the distance between consecutive horizontal braces, starting from the bottom.
  4. (Optional) Set the "Bottom level z-value" to generate the model with an offset in the z-direction.
    Note: You can also move and rotate the jacket to the wanted position in FEDEM through the property fields of the generated subassembly.
  5. (Optional) Toggle on the "Visualize 3D" property to generate the model with 3D visualization in FEDEM.
  6. Push the "Generate" button to create the jacket model in FEDEM.
