Scatter

The Scatter application is used to import and manage wave scatter diagrams for the FEDEM event system.

Blocking can be performed on the scatter data to reduce the number of low probability events (i.e. unnecessary events) in the FEDEM model. A large number of events will take a long time to simulate in FEDEM. By blocking out the low probability events, you can reduce the number of events, and reduce the amount of time needed by the FEDEM solvers.

Paste from Excel

You start by copying and pasting a scatter diagram from Excel (see example below):

  • The top row must contain the Tp-values (midpoint values).
  • The left-most column must contain the Hs-values (midpoint values).
  • The data must contain the probability values for each Hs-Tp combination.

Blocking methods

The "auto-block" and "block selected" buttons (see below) use blocking methods on a range of cells. The blocking methods start by calculating the weighted average, <Hs>, of the Hs values; and the weighted average, <Tp>, of the Tp values. The probability values will be summed and moved to the (<Hs>,<Tp>) cell. Blocking is an effective technique for reducing the number of low-probability events in the FEDEM model.

Auto-block

The Auto-block function merges cells within the same column, i.e. they have same Tp. The blocking method determines the target (<Hs>,<Tp>) cell. The number of cells getting merged into one are limited by the following user settings:
1. Limit probability: The max probability allowed for the resulting cell.
2. Max rows blocked: The total number of rows that that are allowed merged.

Block selected

The selected range of cells are merged into one cell (within the selected range). The blocking method determines the target (<Hs>,<Tp>) cell.

Normalize

The scatter diagram shows probabilities in percentage. Press Normalize in order to normalize the data so that the sum of probabilities becomes 100.

Move cell

Move cell will sum the contents of two selected cells and put the sum in the left or bottom cell. The right or top cell will become blank.

Generate events or output event file

You can Generate Events directly in the currently open FEDEM model, or you can save events as an event input file for FEDEM.
