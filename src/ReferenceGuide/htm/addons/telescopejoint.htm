Telescope Joint

The Telescope Joint application is used to create a telescope joint, that is a cylindrical connection between two beam strings.
The tool contains two separate ways to create such a joint:

Connect Beam strings

"Connect Beam strings" is used when you have an existing pair of beam strings in your model, with overlapping beams in the area you want to create a telescope joint.

How to use it

  1. In FEDEM, select the beams to constitute the "upper barrel" of the telescope joint, by holding ctrl and picking the beams in either the 3D-view or object-tree.
  2. Push the "Get Selection" button for the "Upper Barrel" list.
  3. Verify that the selected beams are listed with their base-IDs and description field.
  4. Repeat the above steps for the "lower barrel"
  5. Push the "Generate Telescope joint" button to create the cylindrical joint connection between the two barrels.

Replace Beam

"Replace Beam" is used when you want to insert a telescope joint as a replacement for an existing beam in a beam string.

How to use it

  1. In FEDEM, select the beam to replace either in the 3D-view or the object-tree.
  2. Push the "Get Selected beam" button.
  3. Verify that the selected beam is shown with its base-ID, description and length.
  4. Specify the outer and inner diameter for both barrels, as well as their respective lengths.
    Note that each of these must be shorter than the length of the beam to replace.
  5. Push the "Generate Telescope joint" button to replace the beam with a telescope joint.
