Addons Overview

FEDEM can use addons for extending the functionality of the modeler. The addons are launched from the Addons menu in FEDEM. The following is a list of addons that has been created for special needs. They are not part of the standard FEDEM distribution.

Beam String Creator

This tool is used to create beam strings as straight lines or in the shape of a catenary curve.

Area: Modeling

Casing FLS

Casing FLS is used to easily calculate fatigue on a large range of beam string systems on the FEDEM platform.

Area: Post-processing

Connect

Connect is an assistive tool to simplify the process of connecting triads, nodes on parts, and ground with a joint.

Area: Modeling

Jacket Creator

Demo application for modeling simple jacket structures.

Area: Modeling

Directional Fatigue Factor

Used to calculate a correction factor for adjustment of fatigue life calculations based on the distribution of wave directions.

Area: Post-processing

Scatter

The Scatter application is used to import and manage wave scatter diagrams for the FEDEM event system.

Area: Modeling

Soil Creator

Soil Creator is used to apply soil-springs to a beam string based on user-defined p-y, t-z and q-z curves.

Area: Modeling

Telescope Joint

The Telescope Joint application is used to create a telescope joint, that is, a cylindrical connection between two beam strings.

Area: Modeling
